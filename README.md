# Character Counter Web App

This is a **Character Counter web application** created for both **UI Essentials** and **JavaScript Essentials** modules.

The goal of this project was to build a complete front-end application using **HTML, CSS, and JavaScript**, while practicing dynamic UI updates and text analysis using JavaScript.

## Features

The application analyzes text in real time and provides:

- Character count
- Word count
- Sentence count
- Approximate reading time
- Optional character limit with visual feedback
- Letter density analysis
- Expandable letter statistics
- Light / Dark theme toggle
- Option to exclude spaces from character count

## built using:

- **HTML** – structure of the web page  
- **CSS** – styling, layout, and theme handling  
- **JavaScript** – text processing, DOM manipulation, and interactivity
